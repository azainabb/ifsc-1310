Thoughts on Progressive Enhancement

It's a philosophy in which content is emphasized and additional features are layered. Progressive enhancement is a strategy in web design that puts emphasis on web content first. Progressive enhancement is important to modern web development so modern browsers can run all required code on all browsers. It allows everyone to access the basic content and functionality of a web page.
